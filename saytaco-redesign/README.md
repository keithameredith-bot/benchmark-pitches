# Say Taco Tequila Bar & Grill — Redesign Concept

Proposed homepage redesign for [Say Taco](https://www.saytaco.com) (Downtown Ocala, FL) by
**Benchmark Web Development**.

- **Live mockup:** `index.html` (open via GitHub Pages)
- Self-contained, mobile-first single page — replaces the current heavy The7 + WPBakery + Slider Revolution build.
- Uses Say Taco's real brand, content, and photography (owner-authorized).
- Menu prices/items shown are representative placeholders for layout; the real build pulls actual menu data as
  searchable text (the current site's menu is image/PDF only).
- Concept/pitch artifact only.

Previews: `desktop-preview.png`, `mobile-preview.png`.
